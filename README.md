# Kalkulator Numerologiczny

Lekka aplikacja webowa do obliczania wartości Gematrikon, Nous, Psyche i Soma dla podanych imion. Wyniki prezentowane są w tabelach, a sumy i kolejne kroki redukcji oznaczane kolorami specjalnych liczb.

## Funkcje

- Wprowadzenie do 5 imion oddzielonych spacjami  
- Wyświetlenie tabeli z wartością Gematrikon, Nous, Psyche i Soma dla każdej litery  
- Podsumowanie (Total) i redukcja (sumowanie cyfr) aż do liczby jednocyfrowej  
- Kolorowanie sum i kroków redukcji, gdy trafiają na liczby:
  - **Mistrzowskie** (zielony)
  - **Doskonałe** (fioletowy)
  - **Fatalne / Diabelskie** (czerwony)
